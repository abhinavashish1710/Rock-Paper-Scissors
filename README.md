# 🎮 Rock Paper Scissors

A simple Rock Paper Scissors game built in Python where the player competes against the computer.

---

## Features

- 🎮 Player vs Computer
- 🤖 Random computer choice
- 🏆 Winner detection
- 📊 Live scoreboard
- 🔁 Play multiple rounds
- ✅ Input validation

---

## Technologies Used

- Python 3
- Random Module

---

## Project Structure

```
Rock-Paper-Scissors/

```

---

## How to Run

Clone the repository

```bash
git clone https://github.com/abhinavashish1710/Rock-Paper-Scissors.git
```

Move into the folder

```bash
cd Rock-Paper-Scissors
```

Run

```bash
python main.py
```
